## **Installation**

⚠️ **System Requirements:**

- [Node.js 16.8](https://nodejs.org/) or later.
- macOS, Windows (including WSL), and Linux are supported.

```bash
npm install
#or
yarn install
```

## **Getting Started**

```bash
npm run dev
# or
yarn dev
```

💚 Open [http://localhost:3000](http://localhost:3000/) with your browser to see the result.
